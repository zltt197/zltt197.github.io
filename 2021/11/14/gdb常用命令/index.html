gdb(with pwndbg)常用命令

2021-11-14
工具使用说明 > pwn工具-网络安全pwn调试器gdbpwndbg

gdb是pwn手们最常用的调试器

断点

下断点

#对函数下断点
b function_name

#对地址下断点
b *address

#下改监控点
watch expr

#下读监控点
rwatch expr

#下读改监控点
awatch expr

#下条件断点
break ... if expr
wateh ... if expr

#设置断点生效次数
ignore bnum count

查看断点

#查看所有断点
i b

#查看指定断点
i b bnum

#查看监控点
i watchpoint

禁用和启用断点

#禁用所有断点
dis

#启用所有断点
en

#禁用指定断点
dis bnum

#启用指定断点
en bnum

删除断点

#删除所有断点
del

#删除单个断点
del bnum

运行与调试

#直接运行
r

#运行到程序入口
start

#继续运行
c

#单步步过
s

#单步步入
n

#跳转运行
jump *address

#运行到函数结束
nextret

信息查看

#查看所有寄存器
i r

#查看所有函数
i fun

#查看内存布局
vmmap

#查看堆块空闲链
bins

#查看堆分配状况
heap

#查看堆分配状况和内容
vis

#查看栈情况
stack
stack num

#查看变量或地址
print expr
print &expr

#查看内存
x/[num][format][add_length] address
#num:查看的单位长度
#format:显示方式
    x    16进制
    d    10进制
    u    10进制无符号
    o    8进制
    t    2进制
    a    16进制地址
    i    指令
    c    字符
    f    浮点数
#add_length:地址单元的长度
    b    单字节
    h    双字节
    w    4字节
    g    8字节
#常见内存读取指令
#反汇编查看指令
x/20i address
#32位16进制数据
x/20xw address
#64位16进制数据
x/20xg address

#查看内存内容和相关信息
tele address
